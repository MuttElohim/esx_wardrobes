# esx_wardrobes
FiveM resource to create wardrobes points
## Install
- Ensure you're using latest ESX Legacy version
- Download into your resources folder
- Add the next line in your **resource.cfg** file
```
ensure esx_wardrobes
```
## Usage
- **/wardrobesmenu**
